# Hebe Goal App 

A Goal App built by Team Hebe-Underdogs

### Version: 1.1.0

### Usage

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:5000
```

### Live Version
[Live Here](https://hebe-goal-tracker-app.herokuapp.com/)

### MongoDB

Open "config/keys.js" and add your MongoDB URI, local or Atlas
